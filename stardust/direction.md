# Direction — CE Credits Online (Active)

## Phrase
"faithful redesign, modernized — single variant, no gate"

## Restatement (dimensional vocabulary)
A **brand-faithful refresh** of an existing, signal-strong commercial education
site for migration to Edge Delivery. The ask moves the **craft / execution**
axis (modernize) while pinning **palette**, **typography family**, **register**,
and **IA** to the captured surface. No rebrand. Single canonical variant.

## Brand signal
`signal-strong` — palette ≥ 3 distinct clustered colors (cream, ink, forest
green, +); named type families captured (Figtree, Poppins). → **Mode A
(brand-faithful)** is the default; no `--rebrand`, no rebrand-trigger phrase.

## Movements
- **craft / execution**: dated Shopify-Dawn → modern editorial (rolled inside Mode A)
- **palette**: pinned (inherited) — cream `#F5F3ED`, ink `#2D2D2D`, forest `#415D43`
- **typography**: pinned family; Poppins substitutes private Mundial (open-license)
- **register**: brand (education / trust-editorial) — unchanged
- **density**: `balanced` (default; multi-audience hard floor → 64px)
- **ia-fidelity**: `verbatim` — a migration reproduces the source; surface modernizes only
- **decade**: 2025-now (Mode A non-locked roll)
- **ground-family**: `cream` (Mode C override — brand ground wins over seed)

## Gaps / questions
None asked — hands-off run, single variant, no gate. Density and ia-fidelity
defaulted deterministically (balanced / verbatim) per the multi-audience floor
and the migration-reproduces-source quality bar.

## Divergence (brand-faithful mode)
```
decade           ✓ rolled    → 2025-now
craft            ✓ rolled    → editorial modernization
register         inherited   → education / trust-editorial
ground-family    inherited   → cream (brand-native, Mode C override)
font deck        inherited   → Poppins/Figtree (Mundial substituted)
palette          inherited   → cream / ink / forest-green 7-color set
```
Brand-faithful inversions: uppercase→mixed-case headings; private Mundial→Poppins;
live storefront chrome→editorial single-CTA.

## Image-reuse contract (Mode A)
Captured images reused at the same semantic position via the source CDN URLs
(or local copies in `stardust/current/assets/`). Hero stays hero (hero.webp),
section photos stay section photos, logo stays logo (ce-logo.png). No
synthesized placeholders; genuine gaps render as placeholder-with-signature.

## Command sequence
extract ✓ → **direct ✓** → prototype (home first, then per-template) → migrate →
deploy/rollout. Single variant A throughout.

## User confirmation
Hands-off run authorized by the migration prompt — no approval gate. Proceeding
straight to prototype.

## Assumptions that defaulted in
- density = balanced (multi-audience hard floor)
- ia-fidelity = verbatim (migration faithfulness)
- motion register = editorial (source hero is static `<img>`)
- surface softened from pure white to warm-white (OKLCH hard rule)
